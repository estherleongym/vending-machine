Once an item is selected and the appropriate amount of money is inserted, the vending machine should return the correct product
● It should also return change if too much money is provided, or ask for more money if insufficient funds have been inserted
● The machine should take an initial load of products and change. The change will be of denominations 1p, 2p, 5p, 10p, 20p, 50p, £1, £2
● There should be a way of reloading either products or change at a later point
● The machine should keep track of the products and change that it contains


Classes:

Product
- have an initial inventory of items
- add to inventory
- have list of items
- each item has a price

Vending_Machine
- view welcome message and prompt to make a selection
- show list of products
- specify product and prompt for amount due
- accept coin
- only accept coins in certain denominations
- request for more coins if insufficient
- accept more coin
- return product


Coin
- have an initial amount of change in denominations of 1p, 2p, 5p, 10p, 20p, 50p, £1, £2
- add amount of change in machine
- have a store of change
- accept coin and add to change (use same method for loading in change)
- request for more money
- give change if too much money was given


Use Cases:

- Welcome message and display list of products
- Initialize with x products and x amount of change
- View products available
- Select a product
- Prompt for amount due and insert coins for product selected
- Receive coins and add to amount of change in machine
- Return product if coins were sufficient and deduct a unit of product
- Request for more coins if insufficient
- Return product and change if coins were more than needed
- Machine: load more change
- Machine: add in inventory
